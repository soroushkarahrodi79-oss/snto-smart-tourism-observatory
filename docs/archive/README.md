# Archivo — material despriorizado

Documentos que **ya no son el eje narrativo** del observatorio pero se conservan
por su valor metodológico y por si se retoma la línea de trabajo.

El observatorio se centra ahora en el **Parque Nacional Sierra de Guadarrama
(PNSG)** como territorio principal. La **Carta Europea de Turismo Sostenible
(CETS)** se mantiene únicamente como **marco metodológico de gobernanza**, no
como candidatura ni tema central. La **Reserva de la Biosfera Sierra del Rincón**
se conserva como piloto de calibración del método (ver registro de territorios
en `src/config/territories.py`).

## Contenido

- **`CETS_Phase1_correspondence.md`** — Tabla de correspondencia CETS Fase I ↔ SNTO
  para la Reserva de la Biosfera Sierra del Rincón. Útil como referencia del marco
  de acreditación EUROPARC y como plantilla si se reactiva la candidatura del Rincón.

> Para reactivar cualquiera de estas líneas, basta con devolver el documento a
> `docs/` y reintroducir su narrativa en README / WHITEPAPER.
