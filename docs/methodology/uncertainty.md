# Uncertainty

## Purpose
Define the current and desired uncertainty posture for SNTO.

## Background
SNTO includes confidence concepts, provenance labels, sensitivity analysis, and trend uncertainty methods.

## Current State
SNTO has strong uncertainty awareness but incomplete uncertainty accounting. The most important missing capability is end-to-end error propagation from satellite input to final recommendation.

## Evidence
The scientific review praised DCS and provenance but noted missing propagation across cloud masking, geometry, indices, model weights, field validation, and final prioritization.

## Recommendations
SNTO should show uncertainty where decisions are made, not only in methodology pages.

## Next Steps
Develop uncertainty ranges for EHS, SCM, TPI, TIS, and final recommendations before institutional adoption.
