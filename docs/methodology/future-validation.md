# Future Validation

## Purpose
Define the validation roadmap required for scientific and institutional credibility.

## Background
The consensus review found field validation and multi-territory replication essential for global reference status.

## Current State
SNTO can justify pilots, not authoritative international deployment.

## Evidence
The international review board concluded SNTO should be recommended to conservation agencies only as a pilot decision-support and research-validation platform.

## Recommendations
Validation roadmap:
1. Execute field verification in the primary territory.
2. Measure false positives and false negatives.
3. Add control-impact field design.
4. Validate across multiple habitats.
5. Replicate in at least three protected areas.
6. Calibrate thresholds and weights with expert and empirical evidence.
7. Publish methods and validation results separately.

## Next Steps
Tie validation milestones to v1.1, v1.2, and v2.0 readiness gates.
