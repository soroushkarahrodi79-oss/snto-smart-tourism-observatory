# Value Proposition

## Purpose
Define the core value SNTO offers.

## Background
Protected areas often lack a defensible way to decide which visitor-facing assets need conservation attention first.

## Current State
SNTO's value is the translation chain: ecological signal -> visitor-pressure inference -> confidence -> priority -> budget -> report.

## Evidence
All major reviews identified this chain as the strongest differentiated contribution.

## Recommendations
Make the value proposition concrete: reduce uncertainty in conservation budgeting and visitor-pressure management.

## Next Steps
Develop value metrics for pilots: faster prioritization, better budget justification, improved inspection targeting, and clearer reporting.
