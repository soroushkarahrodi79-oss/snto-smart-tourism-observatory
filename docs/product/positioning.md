# Product Positioning

## Purpose
Define how SNTO should be positioned in the market.

## Background
The reviews warned that SNTO will lose if it competes as GIS, remote sensing, or generic dashboard software.

## Current State
The recommended category is protected-area decision intelligence.

## Evidence
The CPO review identified the strongest position: "SNTO helps protected-area managers prioritize conservation action on visitor-facing natural assets using satellite evidence, confidence scoring, and budget-ready recommendations."

## Recommendations
Avoid positioning SNTO as:
- generic smart tourism;
- generic observatory;
- GIS replacement;
- BI dashboard;
- remote-sensing platform.

Position SNTO as the decision layer above existing GIS and Earth observation platforms.

## Next Steps
Rewrite future product narratives around buyer pain and decision workflows.
