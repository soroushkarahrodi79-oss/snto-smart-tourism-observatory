# Future Features

## Purpose
Classify future features into core and premium capabilities.

## Background
The CPO review separated core product from premium expansion.

## Current State
Core features:
- asset risk and intervention console;
- confidence-aware recommendations;
- field verification;
- intervention lifecycle;
- reporting exports.

Premium features:
- multi-park benchmarking;
- advanced remote-sensing automation;
- GIS integration;
- field mobile app;
- climate scenarios;
- intervention ROI;
- public transparency portal;
- national agency rollups.

## Evidence
Feature classification comes from the CPO review.

## Recommendations
Do not build premium features until the core decision workflow is validated.

## Next Steps
Map features to v1.2, v2.0, v3.0, and future releases.
