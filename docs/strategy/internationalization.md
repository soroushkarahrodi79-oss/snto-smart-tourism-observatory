# Internationalization

## Purpose
Define conditions for international expansion.

## Background
The consensus review asked whether SNTO could become a global reference platform.

## Current State
International success probability was estimated at 35%. Generalizability is not yet proven.

## Evidence
The scientific review identified missing cross-territory and habitat validation as major blockers.

## Recommendations
Expand only through validated pilots across different protected-area contexts.

## Next Steps
Select pilot territories that test habitat, governance, visitor-pressure, and data diversity.
