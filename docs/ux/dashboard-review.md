# Dashboard Review

## Purpose
Preserve the dashboard-specific critique.

## Background
The dashboard is currently the primary product surface.

## Current State
The dashboard is comprehensive but too dense for daily executive use.

## Evidence
The director review identified home-page candidates: executive risk brief, top priority assets, visitor pressure alerts, budget impact, evidence confidence, and political reporting snapshot.

## Recommendations
Transform dashboard into a decision console. Move methodology to an audit/evidence layer.

## Next Steps
Create a director-grade homepage and analyst-grade drilldowns.
