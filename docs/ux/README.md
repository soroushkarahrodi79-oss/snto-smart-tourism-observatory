# UX Documentation

## Purpose
Document UX, dashboard, information architecture, accessibility, visualization, and UI evolution findings.

## Background
The product, director, and UI reviews found that SNTO shows too much capability at once.

## Current State
SNTO is information-rich but not yet decision-centered enough for daily institutional users.
